# voycar-frontend

### Setup Documentation

```sh
npm create skeleton-app@latest .
# => template: Bare Bones
# => theme: Skeleton
# => other packages: none
# => type checking: Yes, using JavaScript with JSDoc comments
# => additional setup: ESLint, Prettier, Svelte Inspector
npm i
```

### Usage

```sh
# run app
npm run dev
# reload by typing "r" and enter in the terminal
# inspect svelte components with ctrl+alt+shift+i in webapp

# check svelte code for warnings and errors
npm run check-svelte

# check formatting / code style
npm run check-format
# apply configured formatting rules
npm run format
```
